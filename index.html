About me (O sobie)

The purpose of this website is to share my knowledge of basic, academic programming that is still not available on the Internet or in other publications, with complete codes ready to run. Among these several topics, I would like to point four of them here:

- Solving 'n' equations using Cramer's method (in 'C' language) - a purely academic topic, but the program is ready for analysis and extension.
- Electrostatic charge field graphics (in GW-Basic) - for primary and secondary schools, what is not so easy to convey experimentally.
- A two-dimensional graph of a mathematical function (in 'C' language) with the possibility of exchanging many of its parameters and subjecting it to automatic analysis.
- Three-dimensional graph of a mathematical function (in GW-Basic). The program can be used not only for the function: Scanning the seabed or the surface of the famous caves, requires an algorithm that converts each point in space, defined by the image resolution, into a point of a flat screen - it is less than 30 lines of code (lines from 10000 to 10300 of the program XYZ_EN in GW-Basic).
- ...

Once you copy some of my code and even change it to your preferred language, you own it. You have full rights to change that copy to suit your needs and have full control over it.

To be sure that selected part could be a subject of your interest, look into the first file ("before you start") of each of four main topics. You will find there answers your following questions:
- Is it really something that is part of my interest?
- How can I install needed environment if not 'Dev C++' (it is Borland Turbo C++ Version 3.0 or GW-Basic)?
- How can I run the provided below code?
- How can I change input data?

Mainframe - prezentacje (in Polish only)

Since there is a lot of publications on Internet about the Mainframe technology in English, I have decided to do some presentations in Polish only. However, an English reader can find them quite useful because of technical terms that have international meanings. The same time there is a lack of a good and comprehensive written knowledge about the Mainframe in Polish language.
The files below have the extension .PPTX and are accessed only while downloaded. However, they contain full information (because they contain PowerPoint 'Notes').
You can always exchange them to .PDF files. They will only contain slides without 'Notes', but the readability of the files increases.

Ponieważ w Internecie jest dużo publikacji na temat technologii Mainframe w języku angielskim, postanowiłem zrobić prezentację tylko w języku polskim. Jednakże angielski czytelnik może uznać je za całkiem przydatne ze względu na zawartość techniczną, która ma oddźwięk języka międzynarodowego. Jednocześnie brakuje dobrej i kompleksowej dokumentacji na temat Mainframe w języku polskim.
Poniższe pliki mają rozszerzenie .PPTX i są dostępne tylko poprzez ich pobieranie (download). Zawierają one jednak pełne tematycznie informacje (ponieważ mają w sobie 'Notatki' programu PowerPoint).
Zawsze możesz z nich zrobić pliki z rozszerzeniem .PDF. Będą one zawierać tylko slajdy bez 'Notatek' ale czytelność plików będzie lepsza.
